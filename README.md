### Play online

[ggj15-madhouse.herokuapp.com](http://ggj15-madhouse.herokuapp.com/)

## Run locally

You have to have nodejs and npm installed.
Then clone the repo and cd into it.

```
$ npm install
$ coffee server.coffee
```
